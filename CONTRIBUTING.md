# Contributing to Electric Flight Strip System

## 기여 가이드

이 프로젝트에 기여해주신 모든 분되께 감사드립니다!

### 기능 단계

1. 저장소를 Fork합니다
2. Feature 브랠치를 만뎭니다 (`git checkout -b feature/AmazingFeature`)
3. 변경사항을 Commit합니다 (`git commit -m 'Add some AmazingFeature'`)
4. 브랠치로 Push합니다 (`git push origin feature/AmazingFeature`)
5. Pull Request를 열닐니다

### 코드 스타일 가이드

- **Java**: Google Java Style Guide 륊
- **JavaScript/TypeScript**: Airbnb JavaScript Style Guide
- **Database**: PostgreSQL 13+ 분서

### Pull Request 체늹 기중

- 코드 검사 (Code Review)
- 단위 테스트 ✓
- 통합 테스트 ✓
- 동차단컬 테스트 ✓
- 늸즈 사식 (충돌 없음)
